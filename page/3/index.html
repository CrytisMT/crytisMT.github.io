入手浦科特M5S

发表于 2013-02-20   |  

早之前就对SSD心痒痒，无奈一直价格太高，又不想用小容量的。前些时间有款影驰的256G的SSD只要899，本来都付款了，又担心是白片，无奈取消订单。

看样子也没啥降的趋势了，忍不住入了一块浦科特M5S 128G。据说新版的已经和M5P相同的主控了，哈哈，不错。

先来上个图，老笔记本了，只有SATA2，跑分上与SATA3有些差距，实际使用中感觉不太出来。ahci、4k对齐啥的就不说了。

4K有点低。。。更新一下ahci驱动。。

4k有些提升，其他的没什么变化。。由于我的主板是hm55,5系列的由于“节能”的缘故，随机读写和寻址速度被限制了，关闭节能之后再来张图：

呵呵，这下有显著提高了。还可以接着修改节能的相关设置，但是我自己试验没有提示，反而还有下降，不知道怎么回事，用5系主板的可以试试。

附修改方法：（适用于Intel 5系主板，其他的不知道好不好使）

1、进入注册表：

HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\Processor
HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\Intelppm

把这2个下的Start值从3改成4就成了。

2、在注册表里搜索如下注册表项：

06cadf0e-64ed-448a-8927-ce7bf90eb35d

12a0ab44-fe28-4fa9-b3bd-4b64f44960a6

40fbefc7-2e9d-4d25-a185-0cfd8574bac6

4b92d758-5a24-4851-a470-815d78aee119

7b224883-b3cc-4d79-819f-8374152cbe7c

943c8cb6-6f93-4227-ad87-e9a3feec08d1
将attributes制设为0，无需重启。

打开WIN7电源管理，选择高性能（或者其他方案也可以，不过要记得自己修改的是哪个方案，因为修改后，这个方案就成了最不节省能源的方案）然后点击更改计划设置。

接下来选择处理器电源管理，会看到比正常状态多出了很多可以调节的选项，这个就是上半部分工作的效果。

国外论坛原帖建议的此几处设置为：

处理器性能提高阀值：30%
处理系新能降低阀值：85%
处理器性能降低策略：理想的
处理器性能降低阀值：85%
处理器闲置升级阀值：95%
处理器性能核心放置过度利用阀值：95%

不过还可以最大设置为如下方案，我个人使用的就是这个方案。
处理器性能提高阀值：0%
处理系新能降低阀值：0%
处理器性能降低策略：理想的
处理器性能降低阀值：100%
处理器闲置升级阀值：100%
处理器性能核心放置过度利用阀值：100%

设置的时候一定注意设置到接通电源那子项，这样而使用电池的子项保持默认，这样当你使用电池的时候，即使方案是高性能也不会多出多余的消耗。

无需重启，立即生效。

这个方法之所以实用的魅力也在于此，当你接通电源，使用你修改过的方案时，达到最大SSD性能，当你使用电池、或者想节约环保时，可以随意切换到其他方案，只要你没修改过其他方案的CPU选项，那依然是普通状态。
之所为既可解，也可封，比方案一要自在的多。

Discuz! X2.5版QQ互联的QQ注册跳转到完善资料页

发表于 2013-02-07   |  

之前弄论坛，开启了QQ互联，但是2.5版的Discuz！X用QQ登陆后用户组是QQ游客，可以随意发帖，作者显示IP，这就造成了大量垃圾广告。这个方法用QQ登陆后，要么完善资料，要么绑定之前的账号，论坛里好不容易找的，转载过来。

找到source\plugin\qqconnect\connect\connect_login.php文件，搜索代码

$utilService->redirect($referer);

改为

dheader(‘Location:member.php?mod=connect&referer=index.php’);

这样设置之后，及时QQ登陆，也不会再UC和DZ中创建用户，比如完善资料了才会创建。

 

官僚化制度化了的论坛

发表于 2013-02-06   |  

论坛是个好地方，是交流分享的好地方。我个人的理解，论坛只是网络上的一片空间而已，下面是百度百科对论坛的解释：

论坛又名网络论坛BBS，全称为Bulletin Board System（电子公告板）或者Bulletin Board Service（公告板服务）。是Internet上的一种电子信息服务系统。它提供一块公共电子白板，每个用户都可以在上面书写，可发布信息或提出看法。它是一种交互性强，内容丰富而及时的Internet电子信息服务系统。用户在BBS站点上可以获得各种信息服务，发布信息，进行讨论，聊天等等。
几年前或者说现在，很有名的技术论坛有那么三五个，甚至有3F论坛之说—–龙卷风、霏凡、CCF，此外还有Dreamland、深度、嬴政、龙族等。这些论坛在当时都是很多人梦寐以求想要进入的，因为他们大多是不开放注册的。而在今天，风云变迁，有的论坛甚至覆灭然后重建；有的论坛或者在自我封闭的管理理念下或者在相关政策下，越来越不活跃。随着版权意识的越来越强烈，“技术”这一次也慢慢淡化了。

深度转行成为了手机论坛，龙卷风等冷清得很。而CCF貌似就没开放注册过，到现在我还没有账号。我所知道的，唯一活跃点只有霏凡。

很多人在网上骂论坛不开放注册，而我以为不开放或采用自荐、推荐制度是合理的，若像华彩那样，一堆灌水的，会员杂七杂八，的确不堪设想~我有幸在其中一个论坛担任版主，本来我以为论坛是非常和谐的，管理人员之间都是非常友好的，管理是人性化的，昨天的一件事却彻底改变了我的看法。

具体什么事我就不说了，经过这件事我发现论坛其实已经严重社会化了，就是现实社会的一个缩影。一层管一层，各个版区之间水火不容，一句话就有人来找茬，动不动就打到督察院（现实里的法院）。版规甚至甚于法律，版规就是至高无上的，违反了版规甚至没有了有心与无心之说。kill人还分过失不过失呢。

虚拟交易区是最吓人的了，动不动首犯扣论坛币，二犯扣几千论坛币+禁言，三犯永久禁言+不的担保，什么狗屁乱七八糟的！

所以我现在只在自己的版区活动，查查水，评评分就ok，其他的地方只看不说，免得惹祸上身。

纵观论坛历史，其中不乏因为违反版规而被永久禁言（关监狱）的人，其中有的是曾经为论坛做出巨大贡献的人，有的是永远巨大财富的人，身上的论坛货币价值几万RMB。

这个论坛非常鲜明的一个特色是包子，也就是发红包。谁发的红包多、大也就越出名，想感谢一个人就要定向发红包，很像现实社会里的送礼。过生日了要送包，节日了要送包，犯错了要送包，没事了还得送包！

其实我最想说的是只不过是个虚拟的网络，大可不必这样，搞的这么世俗这么社会化。又或许这些人生活里闲的蛋疼，想要扮演另外一种角色？骂来骂去。发来发去，有意思吗？

不光是论坛，现在PT网站也越来越像这种情况迈进了~

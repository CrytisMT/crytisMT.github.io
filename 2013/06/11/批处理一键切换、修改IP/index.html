批处理一键切换、修改IP

发表于 2013-06-11   |  

由于某些需要需要来回切换IP，之前找了几个修改IP的软件，总有些bug，其实一想，还不如用批处理来的快捷方便。下面提供2个批处理用来切换、修改IP。

1.这个批处理可以在两个或多个IP、掩码、网关之间选择切换，没有加入dns的修改，有需要的可以自行添加。代码都很简单，一看就能懂。

上个效果图：

@echo off
color 0a
title IP切换脚本 麦钛蛋www.maitaidan.com
:start
@echo —————修改切换IP by麦钛蛋——————
@echo 请选择ip:
@echo 1.校园网
@echo 2.台式机
@echo —————www.maitaidan.com——————
set /p sLocal=请选择:
if %sLocal%==1 goto home
if %sLocal%==2 goto office
cls
@echo 选择错误，请重新选择
goto start
:home
@echo 正在设置
netsh interface ip set address 本地连接 static 8.8.8.8 255.255.255.0 8.8.8.254 1
goto end
:office
@echo 正在设置网络连接为192.168.137.2
netsh interface ip set address 本地连接 static 192.168.137.2 255.255.255.0 192.168.137.1
:end
echo 设置完成，自动退出

2.第二个批处理可以智能在两个IP之间切换，如果当前是这个IP，则切换为另外一个。

 

@echo off &setlocal enabledelayedexpansion
title IP智能切换脚本 麦钛蛋www.maitaidan.com
color e
Rem ‘/========获取本机的IP地址(局域网)=========/
echo 正在获取本机的IP地址，请稍等…
for /f “tokens=3 skip=2 delims=: “ %%i in (‘nbtstat -n’) do (
set “IP_addr=%%i”
set IP_addr=!IP_addr:~1,-1!
goto :next
)

:next

for /f “tokens=2 delims=:” %%b in (‘ipconfig^|find /i “ip”‘) do set fsip=%%b
echo 当前IP：%fsip%
Set fsip=%fsip: =%

echo ——– ip修改中,请稍后… ———

Rem ‘/=========判断当前IP=============/
if %fsip%==192.168.137.2 (

Rem ‘/=========设为家中IP=============/
netsh interface ip set address 本地连接 static 8.8.8.8 255.255.255.0 8.8.8.254 1

) ELSE (

Rem ‘/=========设为单位IP=============/
netsh interface ip set address 本地连接 static 192.168.137.2 255.255.255.0 192.168.137.1 1

)

ECHO ——– ip修改完成 ——–
ECHO —— 麦钛蛋www.maitaidan.com ——
pause
两个都不错，哈哈。

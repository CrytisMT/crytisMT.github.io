架设自己的PT站教程之安装篇

发表于 2012-11-25   |  

国内现在各大PT站采用的PT程序，大体是以下几种：Nexusphp，晨光版、六维空间版、Xbit，很多版本都是从Nexusphp衍生来的（虽然Nexusphp也是改进人家源码）。个人建站的话，特别是学校，个人感觉六维空间的论坛模式最好，跟Discuz整合的最好了，可惜没开源，只能退而求其次选Nexusphp了。

Nexusphp现在已经开源了，可以在sourceforge.net/projects/nexusphp下载，不过经过我试用之后，发现并不是太好，管理界面全是英文，需要自己汉化。我推荐用MTPTcode.google.com/p/mtpt/，也就是麦田PT采用的程序，也是基于Nexusphp，经过优化之后的版本，比较好用。

网上架设PT的教程不多，给我帮助比较大的是囧啊囧博客。

MTPT的安装跟Nexusphp的安装差不多，而且源码里都提供了步骤：

1、解压源程序到网站根目录。

2、导入数据库文件。

个人感觉最简单的方法就是在phpmyadminl中新建一个mtpt数据库，然后选择mtpt.sql导入即可。或者采用以下命令：

mysql -uroot -p
mysql> create database mtpt;  #你也可以用其他数据库名字替代
quit
mysql -uroot -p mtpt < mtpt.sql

3、修改配置文件。

在config文件下下面有一个allconfig.php 的文件，修改下面的内容：

‘SITENAME’ => ‘站点名称’,

‘BASEURL’ => ‘网站URL’,

‘announce_url’ => ‘localhost/announce.php’,（announce的url地址）

‘mysql_host’ => ‘MySQL主机’,

‘mysql_user’ => ‘数据库用户名’,

‘mysql_pass’ => ‘数据库密码’,

‘mysql_db’ => ‘数据库名’,

【提醒：网址不要加http://】

4、添加管理员账号

首先自己访问网站然后注册一个用户名，接着进入数据库管理（phpMyAdmin），在users表里面找到你注册的用户，编辑它的class属性为16。

5、修改php配置文件（php.ini），确保以下配置选项为‘Off’：

magic_quotes_gpc = Off

magic_quotes_runtime = Off

magic_quotes_sybase = Off
6、修改mysql配置文件，增加连接数：

sudo vi /etc/mysql/my.cnf

max_connections = 1000 #可根据实际需求设置

重启服务：

sudo service mysql restart

7、需要可写权限的目录和文件：很多莫名其妙的错误都是因为权限没修改导致，包括WordPress、Discuz等等

cd /var/www/mtpt/
sudo chmod 777 attachments/ bitbucket/ imdb/cache/ imdb/images/ subs/ torrents/  config/allconfig.php

8、请检查各项服务是否正常启动，修改配置文件后请重启服务：

sudo service mysql restart
sudo service apache2 restart
memcached -u nobody -h 127.0.0.1 -m 64 -d

到此PT初步架设就完成了！
